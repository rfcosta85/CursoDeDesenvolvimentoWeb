Links locais, externos e download de arquivos

Os links ou Hyperlinks são ligações que existem entre páginas. Vamos ver alguns exemplos do uso das âncoras.

Os links locais apontam para documentos que estão no seu próprio site Acesse outra página nesse mesmo servidor clicando no link anterior
