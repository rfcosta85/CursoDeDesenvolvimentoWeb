Links locais, externos e download de arquivos

Os links ou Hyperlinks são ligações que existem entre páginas. Vamos ver alguns exemplos do uso das âncoras.

Os links locais apontam para documentos que estão no seu próprio site Acesse outra página nesse mesmo servidor clicando no link anterior

Também podemos colocar documentos em outras páginas locais. Acesse uma página em outra pasta clicando no link anterior

Também é possível acessar outros sites usando âncoras. Por exemplo, você pode criar um link para acessar o youtube

O link acima faz o usuário ir para outro site e abandona o meu. O ideal é que ele abra outros sites em uma nova aba. Acesse agora o youtube em uma nova aba em branco.

Os links também podem ser usados para fazwer o download de arquivos de outros formatos como ZIP,PDF,RAR, etc. Por exemplo, abra aqui um livro PDF de exemplo ou baixe a versão compactada do livro.
